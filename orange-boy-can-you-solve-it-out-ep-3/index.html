    Man on Graph

    Given a graph of N nodes and M edges. On node i there are men. Then each person moves along 1 edge randomly connected to the node he stands. But if two men walked along the same edge, they’ll fight and both die.

    Given q queries:

    1. “1 x y” set the number of people in X to be Y
    2. “2” find the expected number of people left after moving.

    Proceed the queries.

    Test 1(8%) - the graph is a tree,,q=1,N=1000,M=1000
