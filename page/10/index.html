Orange Boy Can You Solve It Out? Ep. 23 / 神犇，解乎？

思考题 in ancient Chinese

Orange Boy Can You Solve It Out? / 神犇，解乎？

令和元年九月二十一日，余与神犇同饮，乐也。见月色空明，开而无雨，犇曰「如此宴酣之乐，何不以卵玩之以陶冶情操？」，杂然相许。 但见数卵，神犇摆之成堆。共N堆，i堆$A_i$卵耳。犇问之「汝能寻多少L,R整数对，使期间有序，而总亦有序乎？」。难矣，难矣，余思之数日，仍未解！君，解乎？ On 2019 Sept 21, I took part in AGC038 with Orange Boy. Happily, I went out to see the moon, round and clear, the sky, pure and sunny, so Orange Boy said:”Today is such a happy day, why don’t we play pebbles?” Everyone agrees. So there are N piles of pebbles, the i-th pile contains Ai pebbles. Orange Boy asks me:”How many interval L,R can you find, so that after sorting the elements from L to R, the whole array will become sorted too?” That’s hard, so can you solve it?

Example / 例

A={1,3,2,4} Out:4 Explain: L=1,R=1, bad L=1,R=2, bad L=1,R=3, ok L=1,R=4, ok L=2,R=2, bad L=2,R=3 ok L=2,R=4 ok L=3,R=3 bad L=3,R=4 bad

Constriants / 规

Subtask 1(50%):n<=1000 Subtask 2(50%):N<=1000000

Orange Boy Loves Ancient Chinese, so he solved it.

NFLS 2019-9-19 Programming Training

搬运NFLS训练 NFLS Programming Training 2019-9-19 For international readers, please scroll down.

A - 求和(sum.cpp)

Statement

求 $f(n)=a^{n+k}$ 的第q项到p项的和。

Input

n k q p

Output

一个整数，取模$10^9+7$

Example

In: 2 0 2 3 Out: 12

Constriants

For 100%: $1\leq q\leq p\leq 10^9,0\leq n,k\leq 10^9$

B - 倍数(multiple.cpp)

Statement

求一个数n的倍数中二进制下1的最小个数。多测

Input

T n1 n2 … nT

Output

T行。

Example

In: 3 4 7 11 Out: 1 3 2 Explain: $11\times3=33=(100001)_2$

Constriants

For 40%: $1\leq n\leq20$ For 70%: $1\leq n\leq1000$ For 100%: $1\leq n\leq10^6,1\leq T\leq 3$

C - 火星语 (opal.cpp)

Statement

表达式计算。 定义 (a,b)=(a+b)%10000 定义 [a,b]=(a<b?0:9999) 定义 {a?b:c}=(a%2==1?b:c) 求表达式的值。

Input

s

Output

一个整数

Example

In: {[4,3]?(1,2):7} Out: 3

Constriants

Subtask 1: 只有() Subtask 2: 只有()和[] Subtask 3: $1\leqS\leq10^6$

English Version

A - Sum

Statement

Find the sum of the q-th term to the p-th term of $f(n)=a^{n+k}$

Input

n k q p

Output

One integer, module 1e9+7

Example

In: 2 0 2 3 Out: 12

Constriants

For 100%: $1\leq q\leq p\leq 10^9,0\leq n,k\leq 10^9$

B - Multiple

Statement

Find the minimal digit ‘1’ in the multiple of the given integer N. You need to solve the question T times.

Input

T n1 n2 … nT

Output

T lines.

Example

In: 3 4 7 11 Out: 1 3 2 Explain: $11\times3=33=(100001)_2$

Constriants

For 40%: $1\leq n\leq20$ For 70%: $1\leq n\leq1000$ For 100%: $1\leq n\leq10^6,1\leq T\leq 3$

C - Opal

Statement

Define (a,b)=(a+b)%10000 Define [a,b]=(a<b?0:9999) Define {a?b:c}=(a%2==1?b:c) Find the value of a given expression s.

Input

s

Output

One integer

Example

In: {[4,3]?(1,2):7} Out: 3

Constriants

Subtask 1: S contains operation () only Subtask 2: S contains operation () and [] only Subtask 3: $1\leqS\leq10^6$
